# Build Roadmap

## Phase 0: Foundation (Day 1-2)
- Finalize MVP schema.
- Create Supabase project.
- Apply SQL schema + RLS.
- Create Next.js project wiring and env handling.

## Phase 1: Core Product (Day 3-7)
- Auth (magic link + Google).
- Profile creation with generated username.
- Home feed with tabs and category filters.
- Poll detail page.
- Vote flow (insert or update before close).
- Real-time or near-real-time totals refresh.

## Phase 2: Creation + Moderation (Day 8-11)
- Poll submission form.
- AI wording assistant for neutral titles, blurbs, and descriptions.
- Submission queue for admins.
- Approve/reject/merge tools.
- Seed flow for initial UK politics polls.

## Phase 3: Quality + Launch (Day 12-16)
- Rate limits.
- Report flow.
- Legal pages final copy.
- QA and bug fixes.
- Deploy to Vercel and connect production Supabase.

## Phase 4: Post-Launch (Day 17+)
- Analytics instrumentation.
- Improve ranking model for Trending.
- Start comments and media-poll architecture.
